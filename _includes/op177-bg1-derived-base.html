{% include relurl.html %} Applies the EFF file specified by the resource key to the targeted creature(s), if the targeted creature(s) match the value specified by the IDS Entry field, in the specified IDS File.
Known values for IDS File are:
  • 2 ⟶ EA.IDS
  • 3 ⟶ GENERAL.IDS
  • 4 ⟶ RACE.IDS
  • 5 ⟶ CLASS.IDS
  • 6 ⟶ SPECIFIC.IDS
  • 7 ⟶ GENDER.IDS
  • 8 ⟶ ALIGN.IDS
{% capture note %} This effect doesn't add the opcode contained in the EFF file to the targeted creature(s). Instead, the op177 instance itself stays on the target(s) via the usual duration rules, and internally decodes and executes the opcode contained in the EFF file every time the op177 instance is ticked. {% endcapture %} {% include important.html %} {% capture note %} The EFF file is applied regardless of the rolls / immunities of the target(s). {% endcapture %} {% include note.html %} {% capture note %} This effect causes a crash if used by a container.
