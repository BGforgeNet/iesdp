--- n: 146 name: "Spell: Cast Spell (at Creature)" param1: "Casting Level" param2: "Type" bg1: 1 bg2: 1 bgee: 1 iwd1: 1 iwd2: 0 pst: 0 --- Casts the spell specified by the resource key at the level specified by the 'Casting Level' field, in the style specified by the 'Type' field. A 'Casting Level' of 0 casts at the level of the casting creature.
Known values for 'Type' are:
0 Normal Casting at specified level (0 for caster level), non-interruptible
1 Cast Instantly at caster level
2 Cast Instantly at specified level (EE-only)
If the target is a container, the Type parameter is ignored and treated as 0 {% capture %} See opcode #326 for some peculiarities of projectile behaviour for Type 2. {% endcapture %}
