--- n: 379 name: "Curse (Non-cumulative)" param1: "Value" param2: "Irrelevant" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 0 pst: 0 pstee: 1 --- This effect causes a penalty to all saving throws and To Hit rolls in the amount that the Value parameter is set to. For every 1 of the penalty value, all saving throws get a penalty of 1, as well as all die throws and damage from the target. This cannot remove the 'Blessed' status from a target, but can effectively negate it. {% capture note %} This opcode mirrors functionality of PST opcode 203. {% endcapture %} {% include note.html %}
