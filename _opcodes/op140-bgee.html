--- n: 140 name: "Graphics: Casting Glow" param1: "Projectile Index" param2: "Type" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Creates the specified projectile at the targeted creature's location. These effects are usually used for casting glows based on the spell school.
Known values for 'Type' are:
  • 0 => Use 'Projectile Index' projectile value.
  • 1 .. 8 => Crash
  • 9 => PROJECTL[110], Default file: "CGNECROM.PRO"
  • 10 => PROJECTL[111], Default file: 'CGALTERA.PRO'
  • 11 => PROJECTL[112], Default file: 'CGENCHAN.PRO'
  • 12 => PROJECTL[113], Default file: 'CGABJURA.PRO'
  • 13 => PROJECTL[114], Default file: 'CGILLUSI.PRO'
  • 14 => PROJECTL[115], Default file: 'CGCONJUR.PRO'
  • 15 => PROJECTL[116], Default file: 'CGINVOCA.PRO'
  • 16 => PROJECTL[117], Default file: 'CGDIVINA.PRO'
{% capture note %} 'Projectile Index' must be treated as in spells, i.e.: value in PROJECTL.IDS + 1 {% endcapture %} {% include info.html %} {% capture note %} This projectile does not deliver any effects. {% endcapture %} {% include note.html %} {% capture note %} The speed of the projectiles animations ARE affected by STATE_HASTED and STATE_SLOWED of the target. {% endcapture %} {% include note.html %}
