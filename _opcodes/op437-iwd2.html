--- n: 437 name: "Disguise" param1: "Value" param2: "Irrelevant" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 1 pst: 0 pstee: 0 --- If Value is zero, then this opcode changes the animation of the target to cleric (keeps race/gender). This works only for player character animations.
If Value is nonzero, then it will set the animation to that value. A permanent timing will set the base animation ID.
