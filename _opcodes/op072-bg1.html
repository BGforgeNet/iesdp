--- n: 72 opname: "IDS: Set IDS State" param1: "IDS Entry" param2: "IDS File" bg1: 1 bg2: 0 bgee: 0 iwd1: 0 iwd2: 0 pst: 0 pstee: 0 --- Alters the IDS state of the targeted creature(s) specified by the 'IDS File' field, to the value specified by the 'IDS Entry' field.
Known values for 'IDS File' are:
1 Generic
2 Race
3 Class
4 Specific
5 Gender
6 Align
If 'IDS File' is 0, the EA state of the target is altered. The values used do not match ea.ids, use the values below for 'IDS Entry':
0 Friendly (green circle)
64 Neutral (blue circle)
255 Hostile (red circle)
