--- n: 82 name: "Set AI Script" param1: "Irrelevant" param2: "Type" bg1: 1 bg2: 1 bgee: 1 iwd1: 1 iwd2: 1 pst: 0 --- Sets the script type specified by the 'Type' value to the contents of the resource key.
Known values for 'Type' are:
  • 0 ⟶ OVERRIDE
  • 1 ⟶ AREA
  • 2 ⟶ SPECIFICS
  • 4 ⟶ CLASS
  • 5 ⟶ RACE
  • 6 ⟶ GENERAL
  • 7 ⟶ DEFAULT
{% capture note %} This effect can unset scripts from a creature, which can break the game. {% endcapture %} {% include warning.html %} {% capture note %}
  • Effect is always instantaneous/delayed.
  • AREA and SPECIFICS can only be set on non-global creatures (not potential party members or familiars), and are lost upon saving/reload.
