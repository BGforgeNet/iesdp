--- n: 0 name: "Stat: AC vs. Damage Type Modifier" param1: "AC Modifier" param2: "Type" bg1: 1 bg2: 1 bgee: 1 iwd1: 1 iwd2: 0 pst: 1 --- Applies the modifier value specified by the 'AC Modifier' field to the category specified by the 'Type' field.

Known values for 'Type' are:
  • 0 ⟶ All
  • 1 ⟶ Crushing
  • 2 ⟶ Missile
  • 4 ⟶ Piercing
  • 8 ⟶ Slashing
  • 16 ⟶ Base AC setting
    • Sets the targets AC to the value specified by the 'AC Modifier' field. If the targets AC is already 'AC Modifier' or below, this effect will do nothing.

Each modifier type to AC from this opcode is capped to the range [-20, 20]. Each AC type total is capped to the range [-32768,32767].

{% capture note %} IWD1 and PST use a slightly different version. The "Base AC" sets to field - 1 instead.
IWD2 uses different parameters altogether. {% endcapture %} {% include note.html %} {% capture note %} With Timing Modes 1/4/7, alters creature's base stat, leaving behind no removable effect. {% endcapture %} {% include important.html %} {% capture note %}
  • 'Type'=0:
    • Modifies ARMORCLASS.
    • With Timing Modes 1/4/7, alters target's base Armor Class (CRE@0x46, "Natural AC", not "Effective AC")
  • 'Type'=1:
    • Modifies ACCRUSHINGMOD.
    • With Timing Modes 1/4/7, alters target's base Crushing AC (CRE@0x4A)
  • 'Type'=2:
    • Modifies ACMISSILEMOD.
    • With Timing Modes 1/4/7, alters target's base Missile AC (CRE@0x4C)
  • 'Type'=4:
    • Modifies ACPIERCINGMOD.
    • With Timing Modes 1/4/7, alters target's base Piercing AC (CRE@0x4E)
  • 'Type'=8:
    • Modifies ACSLASHINGMOD.
    • With Timing Modes 1/4/7, alters target's base Slashing AC (CRE@0x50)
  • 'Type'=16:
    • Modifies ARMORCLASS.
    • With Timing Modes 1/4/7, alters target's base Armor Class (CRE@0x46, "Natural AC", not "Effective AC"), still subject to "only if better/lower", checked against base value, not current effective value.
