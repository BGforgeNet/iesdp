--- n: 135 name: "Polymorph" param1: "Irrelevant" param2: "Change Specifics" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 0 pst: 1 pstee: 0 --- This effect will polymorph the target into the creature type (i.e. the .cre file name) set by the Resource Key. The Change Specifics parameter sets information about how the polymorph affects the target. The known values for the Change Specifics parameter are 1 for polymorph with creature colouring, spellcasting enabled, and gain none of creature statistics or resistences; and 2 for polymorph with creature statistics only.
