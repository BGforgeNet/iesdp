--- n: 206 name: "Embalm (Non-commulitive)" param1: "Irrelevant" param2: "Embalming Type" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 0 pst: 1 pstee: 0 --- The embalming type parameter determines what affect is given to the target of the effect.
A 0 will give the target 2-8 hit points and +1 AC for the duration of the spell.
Any other value gives the target +2 hit points per level of the caster and +2 AC.
Both effects last for the duration of the spell.
