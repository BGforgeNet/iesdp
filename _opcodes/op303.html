--- n: 303 opname: "Spell Effect: Backstab Every Hit" param1: "Irrelevant" param2: "Stat Value" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 pstee: 1 --- The targeted creature(s) will backstab with every hit, regardless of invisibility or positioning (flanking).

The effect modifies ASSASSINATE to Stat Value. The Stat Value parameter can take any value. If you set it to 0, you will disable the effect (for its duration). If ToBEx is installed or an EE game is used, more finegrained control is available:
  • 0 ⟶ Deactivate effect
  • 1 (bit 0) ⟶ Ignore invisibility and positining requirements (classic mode)
  • 2 (bit 1) ⟶ Ignore invisibility requirement only
  • 4 (bit 2) ⟶ Ignore positioning requirement only
