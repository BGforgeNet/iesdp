--- n: 109 name: "Helplessness Set in Creature Type" param1: "Index Number" param2: "IDS File" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 0 pst: 1 pstee: 0 --- This item will set the 'helpless' status in the type of creature set by the parameters. It's similar to a hold effect. The IDS file is the file to get the reference number from. The Reference number is the creature type to affect.
The following values are valid for the IDS File parameter:
2 is Enemy/Ally (EA.IDS)
3 is General (GENERAL.IDS)
4 is Race (RACE.IDS)
5 is Class (CLASS.IDS)
6 is Specifics (SPECIFIC.IDS)
7 is Gender (GENDER.IDS)
8 is Alignment (ALIGNMEN.IDS)
