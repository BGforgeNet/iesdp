weapprof.2da

This file defines the restriction of number of points that could be put in the given weapon proficiency by class/kit, this also defines the name and description text for the specific weapon class..
The first column is filled by weapon proficiency identifiers names.
Second is the ID number of proficiency
Later there are two columns that could be filled by strrefs from Dialog.tlk indicating name and later description of given proficiency
Then we have several columns labeled by kits/classes names and on the crossing of the specific class/kit and weapon proficiency identifier the number of maximal stars that can be put in this proficiency.
