death.2da

This file defines how NPC reacts on death of another NPC.
The first column and row consist of death variables of NPCs.
The crossings of columns and rows indicate if NPC says anything if defined NPC dies. 1 - says REACTION_TO_DIE_SPECIFIC, 0 - says REACTION_TO_DIE_GENERIC.
A non-numeric value indicates one or more comma-separated sound resrefs that are played back instead.
Only a single reaction will take place when a NPC dies, and specific reaction will override a generic reaction.
