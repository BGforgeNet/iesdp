sndchann.2da

This file defines the volume for various game situations.
The first column consists of the identifiers of various game situations where sound files are used like narrator speech for example.
Second column is filled by volume in % probably.
Third column indicates whether 3d sound is simulated.
Fourth column defines the reverb strength.
