speldesc.2da

This file assigns the text descriptions to spells that could be bought in Temples (strrefs).
The first column is filled by spells SPL files references.
The second column contains strrefs to text descriptions of the spells.
The third column contains sound resrefs that are played back when the spell is applied to the target.
