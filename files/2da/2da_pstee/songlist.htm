songlist.2da

This file assigns the MUS file to certain values and is used by scripting and ARE files to determine what music should be played.
The first column is filled by numbers used by ARE files song sections and scripting commands like PlaySong().
The second one contains of descriptions /identifiers (I have no idea if they can be used instead of values in scripts).
And third one contains of MUS files references that describes what ACM sound files are used and how to play song.
