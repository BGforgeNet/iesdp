startare.2da

This file determines the initial area when the game starts and location of the screen view.
The first column is filled with the identifiers AREA, XPOS and YPOS for normal game mode only. AREA defines the start area, XPOS and YPOS determine the location of the screen view. XP and XP_CAP define the starting experience of the Nameless One and the experience cap respectively.
The second column holds entries specifying values for the first column.
