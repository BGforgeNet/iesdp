tooltip.2da

This file defines the text of the "special powers" extended headers (Haste and Invisibility of the Ring of Gaax, etc...)
First column consists of ITM filenames
The following columns define the strrefs consisting of the name of special abilities.
-1 means that there are no special ability.
NB. Extra columns can be added, and will work correctly for item abilities available via the "Use Item" button.
