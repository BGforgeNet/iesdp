xplist.2da

This file defines the xp value received by the party members for doing specified quests represented by identifiers from the first column.
The second column is as written, comment. the rest columns indicates level when party member receives xp. Column labeled as "0" represents the situation when party members always receive xp after finishing quest.
Prefixing the experience amount with P_ will award the XP to the party, otherwise it will be awarded to each party member.
