itemexcl.2da

This file defines items that cannot be used at the same time; value 0 can be used value 1 cannot be used together.
First column defines the list of ITM resrefs.
2nd column is respectively 1 - to not allow using item with other items from the list, or 0 - to allow the item to be used.
